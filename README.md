# CS546_final_project
Dependencies: 
1. NodeJS
2. MongoDB
3. bcrypt
4. express
5. express-handlebars
6. express-session
7. joi
8. moment
9. mongoose
10. nodemon
11. xss

Steps to run SBOARD:
1. In terminal install libraries with "npm install" command
2. Run "npm run seed" to populate data
3. After installation run the application using "npm start"
